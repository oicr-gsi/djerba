Raw Coverage (Mean):Average number of bases covering each sequenced base. Target coverage is 15,000x for the tumour, and 5,000x for the normal.

Unique Molecular Coverage(Mean):Average number of bases covering each sequenced base after Unique molecule based error suppression. Minimum coverage is 400x for the tumour, and 400x for the normal.

Estimated Cancer Cell Content (%):The inferred tumour purity as determined using ichorCNA

Copy State: Copy state changes were called according to the log2 coverage-ratio, which can be used to calculate an approximate minimum copy number for a diploid tumour with 100% cancer cell content. As the log2 coverage-ratio thresholds scale with tumour purity, absolute copy number may be lower than predicted, in tumours with lower cancer cell content.
