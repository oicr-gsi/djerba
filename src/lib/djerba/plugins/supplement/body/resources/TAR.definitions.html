Copy State: Copy state changes were called according to the log2 coverage-ratio, which can be used to calculate an approximate minimum copy number for a diploid tumour with 100% cancer cell content. As the log2 coverage-ratio thresholds scale with tumour purity, absolute copy number may be lower than predicted, in tumours with lower cancer cell content.
