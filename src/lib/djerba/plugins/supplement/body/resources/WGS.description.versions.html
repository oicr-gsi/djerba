Whole Genome libraries were prepared using the KAPA Hyper Prep kit with DNA extracted from FFPE or fresh frozen tissue (for tumour samples) or buffy coat blood specimens (for matched normal blood samples). Paired-end sequencing was performed using the Illumina ${versions.ILLUMINA_VERSION} technology. Reads were aligned using bwa mem (${versions.BWAMEM_VERSION}) against reference genome ${versions.REFERENCE_GENOME_VERSION} and processed according to GATK best practices, including duplicate marking with Picard (${versions.PICARD_VERSION}), realignment around small insertions and deletions (in/dels), and base recalibration with GATK (v.${versions.GATK_VERSION}). SNVs and in/dels were called using MuTect2 (GATK v.${versions.MUTECT2_VERSION}) and annotated with VariantEffectPredictor (v.${versions.VARIANTEFFECTPREDICTOR_VERSION}) using MANE transcripts (MANE Clinical version 1.0 when available, MANE Select version ${versions.MANE_VERSION} for all other transcripts). Variants were further annotated for oncogenicity and actionability by OncoKB. In cases where OncoKB does not use MANE Select, links in annotation use the corresponding alteration in OncoKB. Copy number variations were called using Sequenza (${versions.SEQUENZA_VERSION}). Microsatellite (MS) Instability status is called using msisensor-pro (${versions.MICROSATELLITE_VERSION}) and a custom list of MS sites created by msisensor-pro for the current reference genome.
