Whole Transcriptome libraries were prepared using the Illumina TruSeq Stranded Total RNA Library Prep Gold kit. Paired-end sequencing was performed using the Illumina ${versions.ILLUMINA_VERSION} technology. Reads were aligned using STAR (${versions.STAR_VERSION}) and gene expression levels quantified using RSEM (${versions.RSEM_VERSION}). Fusions were called using STAR-Fusion (${versions.STARFUSION_VERSION}) and Arriba (${versions.ARRIBA_VERSION}), followed by post processing with MAVIS (${versions.MAVIS_VERSION}) and annotation using OncoKB.
