Whole Transcriptome libraries were prepared using the Illumina TruSeq Stranded Total RNA Library Prep Gold kit. Paired-end sequencing was performed using the Illumina ${versions.ILLUMINA_VERSION} technology. Reads were aligned using STAR (${versions.STAR_VERSION}) and gene expression levels quantified using RSEM (${versions.RSEM_VERSION}). Fusions were called using STAR-Fusion (${versions.STARFUSION_VERSION}) and Arriba (${versions.ARRIBA_VERSION}), followed by post processing with MAVIS (${versions.MAVIS_VERSION}) and annotation using OncoKB and the NCCN Compendium of biomarkers. The NCCN Compendium is a manually compiled list of somatic alterations for reporting in specific cancers. For Plasma Cell Myeloma (PCM) cases, we report t(4;14), t(14;16), t(11;14), and t(14;20) (NCCN Multiple Myeloma version ${versions.NCCN_PCM_VERSION}). For ovarian cancer (OVARY), we report Homologous Recombination Deficiency (HRD) (NCCN Ovarian Cancer version ${versions.NCCN_OVARIAN_VERSION}).
