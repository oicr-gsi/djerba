Whole Transcriptome libraries are prepared using the Illumina TruSeq Stranded Total RNA Library Prep Gold kit with RNA extracted from FFPE or fresh frozen tissue tissue. Paired-end sequencing is performed using the NovaSeq6000 technology. Reads are aligned using STAR (2.7.3a) and expressions quantified using RSEM (1.3.3). Fusions are called using STAR-Fusion (1.8.1) and Arriba (1.2.0), followed by post processing with MAVIS (2.2.6).
