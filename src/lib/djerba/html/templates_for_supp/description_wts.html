Whole Transcriptome libraries were prepared using the Illumina TruSeq Stranded Total RNA Library Prep Gold kit. Paired-end sequencing was performed using the Illumina NovaSeq6000 technology. Reads were aligned using STAR (${constants.STAR_VERSION}) and gene expression levels quantified using RSEM (${constants.RSEM_VERSION}). Fusions were called using STAR-Fusion (${constants.STARFUSION_VERSION}) and Arriba (${constants.ARRIBA_VERSION}), followed by post processing with MAVIS (${constants.MAVIS_VERSION}) and annotation using OncoKB.
