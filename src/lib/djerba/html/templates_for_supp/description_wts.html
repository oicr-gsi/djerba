Whole Transcriptome libraries were prepared using the Illumina TruSeq Stranded Total RNA Library Prep Gold kit. Paired-end sequencing was performed using the Illumina NovaSeq6000 technology. Reads were aligned using STAR (2.7.3a) and gene expression levels quantified using RSEM (1.3.3). Fusions were called using STAR-Fusion (1.8.1) and Arriba (1.2.0), followed by post processing with MAVIS (2.2.6) and annotation using OncoKB.
