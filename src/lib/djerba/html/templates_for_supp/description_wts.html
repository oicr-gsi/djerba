Whole Transcriptome libraries were prepared using the Illumina TruSeq Stranded Total RNA Library Prep Gold kit. Paired-end sequencing was performed using the Illumina NovaSeq6000 technology. Reads were aligned using STAR (${versions.get(constants.STAR_VERSION)}) and gene expression levels quantified using RSEM (${versions.get(constants.RSEM_VERSION)}). Fusions were called using STAR-Fusion (${versions.get(constants.STARFUSION_VERSION)}) and Arriba (${versions.get(constants.ARRIBA_VERSION)}), followed by post processing with MAVIS (${versions.get(constants.MAVIS_VERSION)}) and annotation using OncoKB.
